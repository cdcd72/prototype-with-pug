# Web prototype with Pug

嘗試透過 Pug 建立初步的網站雛形。

## 如何使用

1. 安裝 [Node.js](https://nodejs.org/en) 穩定版本

2. 安裝相依套件

   ```
   npm install
   ```

3. 轉換 Pug 範本為 HTML

   ```
   npm run build
   ```

4. 安裝 [Live Sass Compiler](https://marketplace.visualstudio.com/items?itemName=glenn2223.live-sass)

   - 用意在於將 `.scss` 編譯成 `.css`
